New Agent Mentor Program

This program supports first‑time licensed agents completing the BCFSA Applied Practice Course through supervision from experienced Real Broker mentors.

Eligibility

  • First Time License issued by BCFSA
  • Applied Practice Course started
  • Mentor Agreement completed through onboarding
